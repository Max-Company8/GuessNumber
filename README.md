# Guess Number Game

Guess Number — Угадай число
Простая GUI‑игра на Python с использованием библиотеки tkinter. Компьютер загадывает число от 1 до 10, а игрок должен его отгадать. После каждой попытки игра подсказывает: больше или меньше введённое число по сравнению с загаданным.
## Особенности игры:
* Простой интерфейс на tkinter — не требует дополнительных зависимостей.
* Подсказки после каждой попытки: «Ваше число меньше загаданного» / «Ваше число больше, чем загадано».
* Счётчик попыток — показывает, за сколько шагов игрок угадал число.
* Обработка ошибок ввода: если ввести не число, появится сообщение об ошибке.
* Кнопка выхода с плавным закрытием приложения через 1 секунду после нажатия.
* Минималистичный дизайн с подобранной цветовой палитрой.
## Как играть:
* Запустите игру.
* Нажмите кнопку «Да», чтобы начать угадывать.
* Введите целое число от 1 до 10 в появившемся поле ввода.
* Нажмите «ОК».
* Следуйте подсказкам игры, пока не угадаете число.
* Чтобы выйти из игры, нажмите кнопку «Выйти».
